## Layout Specifications

### Page Dimensions
- Format: Letter (8.5" × 11")
- Margins: 0.75" (all sides)
- Content width: ~6.0"

### Typography
```
Title:           24pt Dark Blue    (#2C3E50)
Section Headers: 16pt Medium Blue  (#34495E)
Subtitles:       11pt Italic Gray  (#7F8C8D)
Body Text:       10pt Dark Blue    (#2C3E50)
Labels:          10pt Bold Medium  (#34495E)

Line Height: 14pt
Paragraph Spacing: 10pt after
```

### Color Scheme
```
Primary Text:     #2C3E50 (Dark Blue)
Headers:          #34495E (Medium Blue)
Subtitles:        #7F8C8D (Gray)
Table Header BG:  #34495E (Dark Blue)
Table Header FG:  White
Table Row BG Alt: #ECF0F1 (Light Gray) / White
Table Borders:    #BDC3C7 (Medium Gray)
```

### Tables
```
Core Numbers Table:
  Columns: [1.5", 1.0", 3.0"]  (Total: 5.5")
  Header: Dark blue background, white text
  Rows: Alternating light gray / white
  Borders: All cells bordered
```

## Text Wrapping & Truncation

### Table Cell Text
- Core numbers descriptions: Truncated to 50 characters max
- Format: "...{description text}..." if longer

### Section Text
- Full interpretations included (no truncation)
- Word-wrapped at ~65 characters per line
- Natural paragraph breaks

### Line Height
- 14pt leading ensures readable spacing
- Prevents text from appearing cramped
- Professional appearance

## PDF Optimization

### File Size
- Typical PDF: 7-8 KB
- Text-only, minimal compression needed

### Generation Time
- <100ms for typical report
- In-memory (BytesIO buffer)
- No disk I/O

### Compatibility
- Standard PDF format
- Readable on any PDF viewer
- Compatible with all browsers

## Scaling & Responsiveness

### Content Adaptation
```
If text longer than expected:
  - Extend to multiple lines
  - Text wrapping automatic
  - Page breaks added as needed

Table columns scale to:

  - Proportional sizing

All text remains readable at 10pt minimum
```
